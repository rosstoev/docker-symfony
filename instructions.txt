- Create app directory
- add .dockerignore file to ignore /.idea and /.git directories

To create symfony project use symfonycli utility
 - command: docker-compose run --rm symfonycli new app --version="6.4.*" --webapp

To execute php bin/console command in the project
 - command: docker-compose run --rm commander command:name

To build images and container for development
 - command: docker-compose up -d server php database

To execute composer command in the project
 - command: docker compose run --rm composer command/name

 Xdebug:
  Use php-fpm/custom.ini file xdebug configuration.
  On PHPSTORM create server with host:localhost, port:used port for connection to nginx server,
  map as aways local/path:server/path. Next in Run -> Edit Configuraton you have to add PHP Remove Debug configuration
  where for server chooce what you configured as a server in options and IDE key as default PHPSTORM(depend of what is configure in browser)

PHP configuration:
  In php-fpm/custom.ini file can change php configuration. To apply changes just restart container:
    docker compose restart php
